About

Our inspiration for this project came from the need to simplify and streamline the management of a small café. Managing menus, user registrations, and orders manually proved to be cumbersome, so we created this application to automate these processes.

This project was developed as part of the Portfolio Project for Holberton School. Learn more about Holberton School.
